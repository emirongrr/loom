:::warning
Loom is pre-audit software. Do not use it to secure production assets.
:::

Loom is not a closed wallet application. It is the account and SDK foundation
below wallets: immutable accounts, Passkeys, bounded permissions, Recovery,
vault policy, keystore direction, and a privacy boundary that requires Provider consent before network access.
