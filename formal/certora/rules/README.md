# CVL Rule Workspace

CVL files will live here once each rule has:

- a plain-language property statement;
- target contract list;
- assumptions and summaries;
- local runtime evidence;
- reviewer notes for expected counterexamples.

The first active rules live in `../specs/LoomAccountAuthority.spec`. Additional
files in this directory may be used for shared rule fragments once they are
supported by a pinned `certoraRun` command.

Do not add placeholder rules that are not run locally.
